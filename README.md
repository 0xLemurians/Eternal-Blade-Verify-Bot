# Eternal Blades Discord Bot

A custom Discord bot built exclusively for the Eternal Blades community.

## Features

### ✅ Verification System
- One-click verification button
- Automatically assigns the Blade Seeker role
- Restricts new members to the Blade Gate until verified

### 🎫 Ticket System
- Dropdown ticket panel
- Support
- Report
- Collaboration
- WL Help
- Private ticket channels
- Staff-only access

### 🛡 Moderation
- Role-based permissions
- Community Manager tools
- Founder tools

### ⚙ Hosting
- Hosted on Railway
- Automatic restart on failure
- Environment variable support

## Tech Stack

- Node.js
- Discord.js v14
- Railway
